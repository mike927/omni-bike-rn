## Export Requirements
* **Strava:** Standard OAuth and API upload for completed sessions.
* **Apple HealthKit:** Export completed sessions using `react-native-health`.
* **.FIT Export:** Generate raw `.FIT` binary files locally to allow power users to export their sessions to other platforms.
